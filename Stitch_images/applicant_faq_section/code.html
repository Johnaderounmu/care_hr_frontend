Frequently Asked Questions

Find answers to common questions about the application process, document uploads, and technical support.

Application Process

How do I apply for a job?

To apply for a job, navigate to the 'Jobs' section, select the desired position, and click 'Apply Now'. You will be guided through the application form.

What documents do I need to submit?

Typically, you'll need a resume/CV, a cover letter, and any relevant certifications or licenses as specified in the job description.

How long does the application process take?

The application itself takes about 15-20 minutes. The time to hear back varies by position, but we strive to respond to all applicants within two weeks.

Document Upload

What file formats are accepted for document uploads?

We accept PDF, DOC, DOCX, and TXT formats for document uploads. Please ensure your files are not password protected.

Is there a size limit for uploaded documents?

Yes, the maximum file size for each document is 5MB. If your file is larger, please try to compress it or save it as a lower-resolution PDF.

Technical Issues

What should I do if I encounter a technical issue?

If you encounter a technical issue, please try clearing your browser cache and cookies first. If the issue persists, contact our support team at support@careconnect.com or call us at (555) 123-4567.

How can I reset my password?

You can reset your password by clicking the "Forgot Password?" link on the login page. You will receive an email with instructions to create a new password.
